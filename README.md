Sistema de Cadastro e Verificação (CLI)
Um projeto interativo construído em Java puro, desenvolvido para praticar e demonstrar os fundamentos da Programação Orientada a Objetos (POO), estruturas de controle e tratamento de exceções. O sistema roda via terminal (CLI) e interage diretamente com o usuário através de menus dinâmicos.

Funcionalidades
Menu Interativo: Sistema em loop contínuo até o usuário decidir encerrar a aplicação.

Validação de Maioridade: Recebe dados do usuário e aplica regras de negócio para verificar a maioridade.

Cálculo de Idade: Utiliza os atributos da classe para calcular matematicamente o ano de nascimento.

Lista de Convidados (Arrays): Capacidade de instanciar múltiplos objetos e armazená-los na memória utilizando Vetores (Arrays), com impressão automatizada usando laços de repetição.

Tratamento de Exceções: Sistema protegido com try/catch contra quebras causadas por entradas inválidas de dados (ex: digitar letras em campos numéricos).

Tecnologias e Conceitos Aplicados
Java (Linguagem principal)

Orientação a Objetos (POO): Criação de Classes, Atributos, Instanciação de Objetos e Métodos.

Estruturas de Controle: Laços de repetição (while, for) e estruturas de decisão (Switch Case Moderno ->, if/else).

Scanner API: Para leitura e processamento de dados via console.

Como Executar
Certifique-se de ter o Java JDK instalado na sua máquina.

Baixe ou clone este repositório.

Compile o código rodando o comando no terminal:
javac Sistema.java

Execute o programa com o comando:
java Sistema
